# mirror_matrix_vertical
---

Escribe una función:

def mirror_matrix_vertical(matrix: list) -> list:

Devuelve una nueva matriz reflejada verticalmente, es decir, el orden de las filas se invierte.
Las filas en sí permanecen sin cambios.
La matriz original no debe modificarse.

Ejemplo:

matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

mirror_matrix_vertical(matrix) →
[
  [7, 8, 9],
  [4, 5, 6],
  [1, 2, 3]
]
