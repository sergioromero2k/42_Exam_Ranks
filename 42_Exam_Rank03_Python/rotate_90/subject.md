# rotate_90
---

Write a function:

```
def rotate_90(matrix: list) -> list:
```

Returns a new matrix rotated 90° clockwise.
The original matrix must not be modified.
The input matrix is assumed to be square.

Example:

matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

rotate_90(matrix) →
[
  [7, 4, 1],
  [8, 5, 2],
  [9, 6, 3]
]
