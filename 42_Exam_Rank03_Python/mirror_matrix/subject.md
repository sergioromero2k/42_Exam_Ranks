# mirror_matrix
---

Write a function:

```
def mirror_matrix(matrix: list[list[int]]) -> list[list[int]]:
```

Returns a new matrix where each row is reversed horizontally.
The original matrix must not be modified.

Example:

matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

mirror_matrix(matrix) →
[
  [3, 2, 1],
  [6, 5, 4],
  [9, 8, 7]
]
