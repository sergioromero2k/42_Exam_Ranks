## max_sliding_window
---
Dado un array de enteros y un tamaño de ventana k, devuelve el máximo de cada ventana deslizante.

### Salida
```
([1,3,-1,-3,5,3,6,7], 3) → [3,3,5,5,6,7]
([1,2,3,4], 1)            → [1,2,3,4]
([1,3,-1,-3,5], 5)        → [5]
([], 3)                   → []
([1,2,3], 0)              → []
([7,7,7,7], 2)            → [7,7,7]
```
