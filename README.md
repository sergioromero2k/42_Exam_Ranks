# 42 School — Exam Rank Collection

A personal repository collecting my solutions and notes for all 42 School exam ranks, from **Exam Rank 02** to **Exam Rank 06**.

Each exam folder is self-contained and includes the exercises, my implementations, and in some cases brief explanations to help understand the logic behind each solution.

If you are a 42 student, use this as a reference — not a shortcut. Read the problem, try it yourself, then compare. That is the only way to actually get better.
