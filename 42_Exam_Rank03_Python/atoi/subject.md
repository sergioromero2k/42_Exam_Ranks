# atoi
---
Write a function atoi(s) that converts a string to an integer without using int() or any built-in conversion function. The function must handle leading whitespace, an optional sign (+ or -), and stop parsing at the first non-digit character.

```
atoi("42")        → 42
atoi("  -42abc")  → -42
atoi("+123")      → 123
atoi("abc")       → 0
```
