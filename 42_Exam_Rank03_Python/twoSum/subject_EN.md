# twoSum
---

Write a function 
```
twoSum(nums: list[int], target: int) -> list[int] 
```
that returns the indices of the two numbers in the list nums such that they add up to the given target.

You may assume that:

* Each input has exactly one solution.
* You cannot use the same element twice.
* The solution should return the indices of the two numbers.

Return the indices as a list [i, j].

### Example:
```
twoSum([2, 7, 11, 15], 9) → [0, 1]
twoSum([5, 5], 10) → [0, 1]
```
