# customSortString
---

Write a function

```
custom_sort(arr: list) -> list 
```
that returns a new list containing all elements of the input list sorted in ascending order. The function should not modify the original list and must use a sorting approach to arrange elements from smallest to largest.

### Example:
```
custom_sort([3, 1, 2]) → [1, 2, 3]
custom_sort([5, -1, 0]) → [-1, 0, 5]
```
