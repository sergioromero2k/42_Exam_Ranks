## Exercises

### String Manipulation
- `alternate_case` ‚Äî Alternate uppercase and lowercase characters in a string.
- `capitalize_words` ‚Äî Capitalize the first letter of each word.
- `whisper_lipher` ‚Äî Different implementations of a custom "whisper" cipher.
- `py_echo_validator` ‚Äî Validate and process echoed input patterns.
- `py_pattern_tracker` ‚Äî Track and analyze patterns in strings.
- `customSortString` ‚Äî Sort a string based on a custom character order.

### Number & Base Conversion
- `atoi` ‚Äî Convert a string to an integer without using `int()`.
- `convert_base` ‚Äî Convert a number (as a string) from one base to another.

### Validation & Logic
- `valid_anagram` ‚Äî Check if two strings are anagrams.
- `brackets` ‚Äî Validate whether brackets are correctly balanced.

### Sorting & Lists
- `sorted` ‚Äî Using `sorted()` with `key=lambda`.
- `merge_and_sort_desc` ‚Äî Merge lists and sort them in descending order.
- `topKFrequent` ‚Äî Return the k most frequent elements from a list.

### Matrix & Rotation
- `mirror_matrix` ‚Äî Return a horizontally mirrored matrix.
- `mirror_matrix_vertical` ‚Äî Return a vertically mirrored matrix.
- `rotate_90` ‚Äî Rotate a matrix 90 degrees.

### Data Structures & Algorithms
- `twoSum` ‚Äî Find two indices whose values sum to a target; returns empty list if none found.
